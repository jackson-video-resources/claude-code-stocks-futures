## What You'll Build

By the end of this course you will have a live, automated trading system that:

- Pulls real-time price data from Alpaca for any stock, futures contract, or Forex pair
- Detects entry signals using your own strategy logic (EMA crossovers, RSI, mean reversion, or any rules you define)
- Sizes positions automatically to a fixed risk percentage per trade
- Places and manages orders on Alpaca without you touching anything
- Runs 24/7 on a server via PM2, restarts itself after crashes, and sends Telegram alerts when it trades
- Retrains its own parameters using an automated ML loop — you set it running, it improves itself overnight
- Scales to multiple markets and multiple strategies running in parallel

You write no code. You describe what you want in plain English. Claude Code builds it.
